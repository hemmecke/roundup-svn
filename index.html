Roundup Issue Tracker Mirror

Roundup is a simple-to-use and -install issue-tracking system with command-line, web and e-mail interfaces.

Roundup officially source code is at maintained in an SVN repository (see http://www.roundup-tracker.org/code.html

This roundup-svn repository at github is a live mirror of the complete official Roundup SVN repository.

Whenever someone commits to the official SVN repository, a commit email triggers an update of the roundup-svn GitHub repository.

For more information of how to use this mirror in order to get the full power of GIT and still being able to commit back to the SVN repository, look at to https://sites.google.com/site/hemmecke/roundup-svn.

This mirror is an offer to developers who prefer git over svn.
